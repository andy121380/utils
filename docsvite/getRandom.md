## getRandom :tada: :100: 
均衡获取指定范围的随机整数
#### 参数 
- min `number = 0` 范围最小整数
 
- max `number = 1` 范围最大整数
 
#### td.ts
::: info
`getRandom(min?: number, max?: number): number`
:::
#### 返回 
- `number` 
::: tip
随机整数
:::
#### 实例 
均衡获取0或者1的数


```ts
getRandom()
```
均衡获取1或者10的数


```ts
getRandom(1, 10)
```
